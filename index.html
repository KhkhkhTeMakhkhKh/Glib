    Welcome to Glib

    This library simplifies usage of various APIs available in C#, such as XNA, Windows Forms, and conversions to common types, such as integer. For example, to parse a string as an integer, you can just run: "Your string".ToInt().

    Building from source

    To build this library from source, you need a C# compiler at least (I recommend getting the IDE Visual C# express 2010) and, to build the full library (including XNA components), XNA game studio 4.0 or later. This project includes a Visual Studio C# Express 2010 solution file. To not include XNA components, delete the XNA folder in your copy. Be sure not to include that change in a pull request!
